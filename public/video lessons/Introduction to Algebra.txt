Let's start with a simple scenario. Imagine you have some marbles in a bag, and you know that if you add 5 more marbles, you'll have 12 marbles total. But you don't know how many marbles you started with. This is where algebra comes in!

In algebra, we use letters to represent numbers we don't know yet. We call these letters "variables." The most common variable is "x," but we can use any letter we want - like y, z, a, b, or even more creative choices.

So in our marble example, we could say:
x + 5 = 12

Where x represents the number of marbles we started with. Our goal in algebra is to figure out what value x must have to make this equation true.

To solve this, we need to "isolate" x - that means we want to get x all by itself on one side of the equation. We do this by performing the opposite operation. Since we're adding 5 to x, we'll subtract 5 from both sides:

x + 5 - 5 = 12 - 5
x = 7

So we started with 7 marbles! Let's check: 7 + 5 = 12. Perfect!

The key principle in algebra is that whatever we do to one side of an equation, we must do to the other side. This keeps the equation balanced, like a perfectly balanced scale.

Let's try another example. Suppose you're saving money, and you know that if you multiply what you have by 3, you'll have $15. How much do you have now?

We can write this as:
